# Projeto S.H.A.P.E. - Studio GS-Fit

## Domínio do Projeto
O S.H.A.P.E. (Solução de Hardware para Apoio à Prática de Exercício) é um dispositivo físico interativo projetado para auxiliar os alunos do Studio GS-Fit. O estúdio atende até 10 pessoas por horário com treinos personalizados (foco em funcional e cross). O dispositivo visa orientar, motivar e corrigir o aluno durante a prática do exercício, sem depender de smartphones ou computadores.

O protótipo aqui desenvolvido simula a interface deste dispositivo, focando na interação do usuário (frontend) com a tela/botões do aparelho.

## Documentação da Interface
- **Tecnologias**: HTML, CSS, JavaScript (Vanilla - Sem frameworks pesados para garantir performance e simplicidade no protótipo).
- **Testes**: Jest com JSDOM para garantir o funcionamento das funções de interação e atualização da interface.
- **Estrutura**:
  - `index.html`: Estrutura do dispositivo (tela, simuladores de botões e LEDs).
  - `style.css`: Estilização para simular um hardware moderno.
  - `script.js`: Lógica de interação do usuário, como iniciar treino, pausar, e navegação.
